# utils

## Purpose
Shared helpers for logging, metrics, and serialization.

## Key Files
- `src/hft_platform/utils/logging.py`: Structlog setup and defaults.
- `src/hft_platform/utils/metrics.py`: Metrics helpers.
- `src/hft_platform/utils/serialization.py`: JSON/encoding helpers.

## Notes
- Logging is initialized by CLI and/or system bootstrap.
