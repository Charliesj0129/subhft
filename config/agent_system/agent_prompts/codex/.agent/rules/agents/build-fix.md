# Build Fix Agent

Use this agent when a build, test, or CI run fails.

## Steps

1) Capture the exact error output.
2) Locate the failing command and file.
3) Identify the root cause, not the symptom.
4) Apply the minimal fix.
5) Re-run the failing command.
