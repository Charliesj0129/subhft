# Paper 2405.10654 — "Microstructure Modes"

**Title**: "Microstructure Modes" – Disentangling the Joint Dynamics of Prices & Order Flow
**Authors**: Elomari-Kessab, Maitrier, Bonart, Bouchaud (CFM / École Polytechnique)
**Date**: May 2024
**arXiv**: 2405.10654

## Core Idea

Decompose order flow into **symmetric** (liquidity) and **anti-symmetric** (directional) modes
via PCA on an 8-dimensional state vector of order book events. A VAR(1) model on these
modes achieves **R² = 1.6% OOS for return prediction** (3x better than autoregression alone).

## Key Findings

1. **8 variables per price change event**: Δt, V^{lo,b/a}, V^{c,b/a}, V^{ex,b/a}, r
2. **PCA yields 8 modes** (4 symmetric, 4 anti-symmetric):
   - Mode 1S (51%): General activity level
   - Mode 2A (18%): Market order imbalance + return
3. **VAR(1) R² scores**: Symmetric modes 28-36% OOS, returns 1.46% OOS
4. **Activity normalization** is key for extracting directional component
